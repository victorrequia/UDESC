# Simulated annealing

### Objetive: Solve 3SAT problem (listed on ./cases) using simulated annealing aproach.

### Dependencies

- Python 3.11.5
- numpy 1.21.2
- matplotlib 3.4.3

### Installing

```bash
pip install -r requirements.txt
python3 main.py
python3 benchmark.py # Run 10x all cases and save results on ./rs and ./ra folders
```
